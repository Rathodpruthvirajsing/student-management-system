SCENARIO 1: Wrong Dashboard Access
─────────────────────────────────
  Student tries to access: /admin_dashboard.php
                                  │
                                  ▼
                        [dashboard.php - session check]
                                  │
                    $_SESSION['role'] !== 'admin'? YES
                                  │
                    Is role === 'student'?
                                  │ YES
                                  ▼
                    header("Location: student_dashboard.php")


SCENARIO 2: Session Expired
──────────────────────────
  User inactive for 30+ minutes
                                  │
                                  ▼
                    [Any protected page - session.php]
                                  │
                    time() - last_activity > 1800?
                                  │ YES
                                  ▼
                    session_destroy()
                                  │
                                  ▼
                    header("Location: home.php?msg=Session+expired")
                                  │
                                  ▼
                    [home.php displays message]


SCENARIO 3: Not Logged In
────────────────────────
  User tries to access protected page directly
                                  │
                                  ▼
                    [Any protected page - session.php]
                                  │
                    isset($_SESSION['user_id'])?
                                  │ NO
                                  ▼
                    header("Location: home.php")


SCENARIO 4: Login Failure
────────────────────────
  Wrong email or password
                                  │
                                  ▼
                    [auth/login.php - verify]
                                  │
                    password_verify() fails?
                                  │ YES
                                  ▼
                    header("Location: index.php?error=Invalid+credentials")
